# IssueSherlock

AI-powered civic issue detective built for Vibe2Ship Hackathon 2026.
Uses a 4-agent Gemini pipeline to analyze, investigate, and autonomously 
escalate unresolved community issues.

## Live Demo
https://issuesherlock-268865731349.us-central1.run.app

## Built With
- Python + Flask
- Gemini 1.5 Flash API
- Firebase Firestore
- Google Cloud Run
- Leaflet.js + OpenStreetMap
